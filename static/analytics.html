Analytics Dashboard

Дашборд аналитики использования оркестратора GraceKelly.

Что показывает

  • Запросы по моделям — распределение выполнений между доступными моделями
  • Успешность — доля завершенных и ошибочных шагов
  • Время отклика — среднее время ответа для каждой модели
  • Топ моделей — рейтинг моделей по успешности
  • Ошибки — количество неуспешных шагов в агрегированной статистике

Источник данных

Все данные берутся из маршрута /api/v1/analytics. Дашборд автоматически обновляется каждые 30 секунд.

Фильтры

Можно выбрать период (сегодня, неделя, месяц, всё время) и конкретную модель для детального анализа.
