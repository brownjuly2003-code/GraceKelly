English Practice

Тренажёр разговорного английского с AI-анализом ваших ответов.

Как это работает

Вы ведёте диалог с AI на выбранную тему. AI играет роль собеседника и после каждого вашего ответа анализирует:

  • Grammar — грамматические ошибки с объяснением правила
  • Vocabulary — более естественные альтернативы для ваших фраз
  • Fluency — насколько ответ звучит как речь носителя языка
  • Suggestions — конкретные улучшения формулировок

Голосовой ввод

Поддерживается Web Speech API — можно говорить в микрофон, и речь автоматически преобразуется в текст. Это помогает тренировать именно разговорную речь, а не письмо.

Модели

Используется одна из моделей Perplexity Pro (по умолчанию Claude Sonnet 4.6) через браузерную автоматизацию. Модель получает системный промпт с инструкцией быть терпеливым учителем английского.
