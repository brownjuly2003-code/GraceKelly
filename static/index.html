GraceKelly Orchestrator

Инструмент для минимизации ошибок LLM через множественный запрос разным моделям Perplexity Pro.

Паттерны

  • SONAR — быстрый ответ через Sonar (собственная модель Perplexity с поиском)
  • SINGLE — один запрос к выбранной модели (GPT-5.4, Claude Sonnet 4.6, Gemini 3.1, Grok 4.1 и др.)
  • DUAL — два запроса к двум разным моделям, результаты рядом для сравнения
  • FIVE MODELS — 5 моделей по одному запросу каждой, все ответы параллельно
  • FIVE MODELS COMPARE — 5 моделей + 6-й вызов Claude для анализа расхождений между ответами
  • CONSENSUS — 5 моделей × 3 вариации промпта = 15 ответов → Mistral embeddings → кластеризация → синтез
  • MAXIMUM — как CONSENSUS, но с итерациями до достижения консенсуса 95%+

Как работает CONSENSUS

1. PLANNER обогащает задачу контекстом и уточняющими вопросами

2. Каждая из 5 моделей получает 3 варианта промпта (нейтральный, критический, креативный)

3. 12 ответов преобразуются в эмбеддинги (Mistral) и группируются в кластеры

4. SYNTHESIZER объединяет ответы из доминирующего кластера в финальный ответ

Дополнительные функции

  • Голосовой ввод (Web Speech API)
  • Прикрепление файлов к запросу
  • История тредов (сохранение и переключение)
  • Отправка результата в Telegram
  • Скачивание ответа в файл
