# Batch 101-b — Unblock `/api/v1/smart` default (minimum viable + optional extend)
Date: 2026-04-23
Tier: XS (Phase A only) → S (if Phase B)

## Bug

`POST /api/v1/smart` с prompt без поля `model` отдаёт:
```
{"answer":"[provider_unavailable] Mistral API key is not configured.", ...}
```
Причина: `Field(default="mistral-small")` в 7 request schemas + ошибочно подключённый `MistralApiAdapter` как LLM backend. Mistral правомерен в проекте **только** как embeddings для consensus (`core/embeddings.py`, аналогично reference-проекту `D:/Perplexity_Orchestrator2/orchestrator/mistral_embeddings.py`) — не как LLM.

## Goal

`POST /api/v1/smart {"prompt":"2+2="}` в `GRACEKELLY_EXECUTION_PROFILE=dry-run` возвращает HTTP 200 без строки `provider_unavailable`/`Mistral API key` в `answer`. `model_id` в response ≠ `mistral-small`.
